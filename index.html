ABOUT MOLE BATTLE

The game is set in a farm, where the main character are moles. The mole attack is based on the most common vegetables that we can find in an allotment, like tomatoes, onions, aubergins, peppers and carrots.

FONDAMENTALS ASPECTS

The basic idea of the game is the same from Worms, where the characters throw objects in order to destroy the other team members. This characters are moles and the objects are vegetables.


Therefore, this game is considered a strategy game based in turns. Every player has a limited amount of moles to play with, each mole has special features which makes a difference between each other. Each player has an inventory which can be filled with vegetables before the battle begins.


The main objective is to destroy all the enemy moles. The first to kill the other's moles wins. Your moles can be damaged by your teammates, so be carefull!
